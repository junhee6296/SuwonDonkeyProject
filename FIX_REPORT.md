# 수정 보고서

## 반영된 요청

1. WinForms 디자이너 흰 화면 문제 해결
   - 컨트롤을 `Form1.Designer.cs`의 `InitializeComponent()` 안에 실제 배치했습니다.
   - 생성자에서는 런타임 전용 초기화만 수행합니다.

2. 이상 주행 자동 탐지 기능 추가
   - `user/angle`을 기준으로 이동평균과 표준편차 기반 변동성 밴드를 계산합니다.
   - 밴드 이탈 또는 급격한 조향 변화가 있는 프레임을 이상 후보로 표시합니다.
   - 목록, 그래프, 타임라인, 필터, 다음 이상 탐색 버튼을 추가했습니다.

3. 이미지 일부 가리기 / 바꾸기 기능 추가
   - 이미지 위에서 드래그한 영역을 마스크 색상으로 가릴 수 있습니다.
   - 선택 영역을 외부 이미지로 교체할 수 있습니다.
   - 편집 전 원본은 `data/_edited_backup`에 백업합니다.

4. 삭제 복구 유지
   - catalog 삭제 시 이미지는 `data/_deleted_backup`으로 이동합니다.
   - 직전 삭제 1회는 `삭제 취소`로 복구됩니다.

## 삭제한 필수 파일

필수 소스 파일은 삭제하지 않았습니다. 공유/제출에 불필요한 개인 환경 파일과 빌드 산출물만 제외했습니다.

제외 대상:
- `.vs/`
- `bin/`
- `obj/`
- `TeamApp.csproj.user`
- `TeamApp.slnx`

## 빌드 대상

- `net8.0-windows`
- Visual Studio 2022 + .NET 8 Desktop Development workload 권장
