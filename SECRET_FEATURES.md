# 추가된 비밀 기능

## 1. 시계열 변동성 기반 이상 주행 자동 탐지

데이터를 불러오면 `user/angle` 시계열을 자동 분석합니다. 각 프레임 주변의 이동평균과 표준편차 기반 변동성 밴드를 계산하고, 조향각이 밴드를 크게 벗어나거나 직전 정상 패턴에서 급격히 튀는 구간을 이상 주행 후보로 표시합니다.

표시 위치:
- 프레임 목록: 빨간 글씨와 `!` 표시
- 그래프: 이동평균/밴드와 빨간 세로선·점 표시
- 타임라인: 빨간 마커 표시
- 필터: `이상 주행 후보만 표시`
- 버튼: `다음 이상`으로 후보 프레임을 순차 탐색

## 2. 이미지 일부 가리기 / 바꾸기

이미지 위에서 드래그한 사각형 영역을 다음 방식으로 편집합니다.

- 검정/흰색/회색으로 가리기
- 선택 영역 평균색으로 자연스럽게 가리기
- 사용자가 선택한 다른 이미지로 해당 영역 교체

편집은 버튼 클릭 즉시 저장되며, 원본은 `data/_edited_backup`에 보관됩니다.

## 3. 디자이너 편집 가능 구조

기존처럼 런타임에서 컨트롤을 동적으로 만드는 방식이 아니라, 버튼과 그룹박스를 `Form1.Designer.cs`에 실제 배치했습니다. 따라서 Visual Studio의 `Form1.cs [디자인]` 화면에서 흰 화면이 아니라 실제 UI가 보이고 위치 조정도 가능합니다.
