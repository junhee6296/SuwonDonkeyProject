# Responsive UI / Batch Edit / Training Builder Fix Report

## 1. 화면 확대/축소 유연성
- 런타임 `ApplyResponsiveLayout()`을 추가하여 창 크기 변경 시 좌측 프레임 목록, 중앙 이미지/편집 영역, 우측 필터/이상탐지/학습/로그 영역을 비율 기반으로 다시 배치합니다.
- `MinimumSize`를 낮추고 `AutoScroll`을 활성화하여 작은 화면에서는 비율 축소 + 스크롤 보정이 되도록 했습니다.

## 2. 버튼 및 문구 재조정
- 상단 `데이터 폴더 열기` 버튼을 `선택한 폴더 경로 확인`으로 변경하여 `폴더 열기`와 의미가 헷갈리지 않게 했습니다.
- `angle = 0 제외` 문구를 `angle=0 제외`로 줄이고 필터 영역을 재배치하여 `최소/최대` 라벨과 겹치지 않도록 했습니다.

## 3. 다중 선택 / 삭제 표시 / 이미지 편집 개선
- 프레임 목록을 `CheckedListBox`로 변경했습니다.
- `전체 선택`, `체크 해제` 버튼을 추가했습니다.
- 체크된 여러 프레임을 기준으로 이미지 영역 가리기, 이미지 영역 교체, 삭제 표시, 삭제 복구, 원본 복구가 가능하도록 수정했습니다.
- 삭제는 실제 이미지/catalog 삭제가 아니라 UI 표시 상태로 처리합니다.
  - 삭제 표시: 빨간 배경
  - 이미지 교체/편집: 초록 배경
  - 이상 후보: 빨간 글씨
- 삭제/편집 상태는 `data/_ui_marks.json`에 저장되어 앱을 다시 열어도 유지됩니다.
- 필터에 `삭제 표시만`, `교체/편집만`을 추가했습니다.

## 4. AI 학습 실행 개선
- `학습 명령 생성` 창을 추가했습니다.
- 학습 데이터 범위 선택:
  - 이상치 포함 전체 데이터 학습
  - 이상치 제외 학습
  - 데이터 필터링 선택군만 학습
  - 선택군에서도 이상치 제외 옵션
- 실행 환경 선택:
  - Windows 환경 빌드
  - WSL 환경 빌드
  - VirtualBox 환경 빌드
- 환경에 따라 Windows 경로, WSL `/mnt/c/...` 경로, VirtualBox `/media/sf_...` 경로를 자동 변환합니다.
- VirtualBox는 SSH 명령을 자동 생성합니다.
- 선택된 학습 범위는 `data/_training_sets/...` 폴더로 catalog/images를 내보낼 수 있게 했습니다.
- `추가/보정 인자` 입력칸을 추가하여 `--type linear` 같은 학습 보정 옵션을 붙일 수 있습니다.
